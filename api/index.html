    Introduction

    Yoga is a cross-platform implementation of flexbox. Here you will find information on the Yoga API, along with Yoga usage scenarios and examples.

    API

    Yoga has a rich API available in 4 languages: C, Java, C# and Objective-C.

    Data Structures

    The following are the key data structures that are used in the calls to the API functions provided by Yoga.

    Examples

    Wrap Content

    By default align-items has a value of stretch. This is usually a good default but can become confusing when you want your container to wrap to the size of its children.

    Say you have something like this.

    But you would like the black container to wrap to the size of the red square. In that case you can adjust the align-self property of the container.

    // White container
CSSNodeRef root = CSSNodeNew();
CSSNodeStyleSetWidth(root, 600);
CSSNodeStyleSetPadding(root, CSSEdgeAll, 50);

// Black container
CSSNodeRef child = CSSNodeNew();
CSSNodeStyleSetPadding(child, CSSEdgeAll, 20);
CSSNodeStyleSetAlignSelf(child, CSSAlignFlexStart);
CSSNodeInsertChild(root, child, 0);

// Red box
CSSNodeRef item = CSSNodeNew();
CSSNodeStyleSetWidth(item, 100);
CSSNodeStyleSetHeight(item, 100);
CSSNodeInsertChild(child, item, 0);

    // White container
CSSNode root = new CSSNode();
root.setWidth(600);
root.setPadding(Spacing.ALL, 50);

// Black container
CSSNode child = new CSSNode();
child.setPadding(Spacing.ALL, 20);
child.setAlignSelf(CSSAlign.FLEX_START);
root.addChildAt(child, 0);

// Red box
CSSNode item = new CSSNode();
item.setWidth(100);
item.setHeight(100);
child.addChildAt(item, 0);

    // White container
var root = new CSSNode();
root.StyleWidth = 600;
root.SetPadding(CSSEdge.All, 50);

// Black container
var child = new CSSNode();
child.SetPadding(CSSEdge.All, 20);
child.AlignSelf = CSSAlign.FlexStart;
root.Insert(0, child);

// Red box
var item = new CSSNode();
item.StyleWidth = 100;
item.StyleHeight = 100;
child.Insert(0, item);

    // White container
UIView *root = [UIView new];
[root css_setUsesFlexbox:YES];
[root css_setWidth:600];
[root css_setPadding:50 forEdge:CSSEdgeAll];

// Black container
UIView *child = [UIView new];
[child css_setUsesFlexbox:YES];
[child css_setPadding:20 forEdge:CSSEdgeAll];
[child css_setAlignSelf:CSSAlignFlexStart];
[root addSubview:child];

// Red box
UIView *item = [UIView new];
[item css_setUsesFlexbox:YES];
[item css_setWidth:100];
[item css_setHeight:100];
[child addSubview:item];

    Centered Content

    Centering content is done with a mix of the align-items and justify-content properties. align-items can center an items in the cross axis of a container and justify-content can center an item in the main axis of a container.

    Assuming you use the default flex-direction of column you can center an item (red) horizontally within a container (black) bu using align-items: center.

    If you want to instead center that item in the vertical direction you can use justify-content: center. Remember that this will center all items in that container so if you want your item to be right in the center you can only have one child in your container.

    Putting these properties together we can center an items inside a container both vertically and horizontally.

    // Black container
CSSNodeRef root = CSSNodeNew();
CSSNodeStyleSetWidth(root, 600);
CSSNodeStyleSetHeight(root, 600);
CSSNodeStyleSetAlignItems(root, CSSAlignCenter);
CSSNodeStyleSetJustifyContent(root, CSSJustifyCenter);

// Red box
CSSNodeRef item = CSSNodeNew();
CSSNodeStyleSetWidth(item, 100);

    Button Bar

    Buttons bars are common in layouts at facebook. A button bar is a horizontal bar of equally size buttons. The footer of a news feed story is a great example of this. Simplified it look something like this.

    The red squares represent buttons which typically contain text and perhaps an icon. Even though the text is different in each button we want every button to be of equal size and their total size should make up the full size of the container. This is easily done in Yoga using flex-grow and flex-basis. flex-basis tells the layout that it should disregard the size of the item and use the provided flex-basis instead, this is how we work around the issue of the texts in each button being of different size. flex-grow takes any remaining space and distributes it among the children of a container.
